How to set up Ollama — free, runs on your machine
01
Install Ollama Download and install from ollama.com — available for Mac, Windows, Linux
02
Start Ollama Open the Ollama app — or run in your terminal: ollama serve
03
Pull a model (once) In a new terminal tab, download the model you want: ollama pull llama3.2 Other options: mistral · gemma2:9b · qwen2.5:14b
04
Select model above & hit Build No API key needed — ProductOS connects to localhost:11434
Only works when ProductOS is running locally on the same machine as Ollama. Quality varies by model — Llama 3.2 is a good starting point.
